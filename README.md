# BetterCalcC
A simple calculator program written in C, improved from the original CalculatorC.

- Asks the user for how many **numbers** are going to be entered.
- Asks the user for which **operation** to use `[1]+ [2]- [3]* [4]/`.
- Asks the user to **input** the numbers one at a time.
- Prints the **answer**.

## Requirements
- GCC (or any C compiler)

## Installation
Clone the repository and install:
```bash
git clone https://github.com/Pesvoh/BetterCalcC.git
cd BetterCalcC
gcc Calculator.c -o Calculator
sudo cp Calculator /usr/local/bin/
```
Then run from anywhere:
```bash
Calculator
```

## Uninstall
```bash
sudo rm /usr/local/bin/Calculator
```
